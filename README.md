# AI Content Studio – Frontend

AI Content Studio is a full-stack AI-powered platform for generating and managing AI-based content and images.  
This repository contains the frontend application built with React and Vite.

🌐 **Live Demo:**  
https://generator-cms-frontend.vercel.app/

---

## 🚀 Features

- User Authentication (Login / Signup)
- Protected Routes
- AI Content Generation (Rewrite, Expand, Shorten, Articles, SEO)
- AI Image Generation with Custom Resolutions
- View Content & Image History
- Fully Responsive UI with Tailwind CSS

---

## 🛠 Tech Stack

- React 19
- Vite
- React Router
- Tailwind CSS
- Axios
- React Hook Form
